# FoodSpot 🍕
A responsive restaurant landing page built with vanilla HTML and CSS. FoodSpot showcases a fictional restaurant's menu, kitchen experts, and customer testimonials in a clean, single-page layout with smooth hover interactions and a mobile-friendly design.

**Live Demo:** [https://jenidub.github.io/ct-foodspot-website/](https://jenidub.github.io/ct-foodspot-website/)

## Overview
This project was built as part of the **Coding Temple Fullstack Certification Program** program to demonstrate core HTML and CSS fundamentals.

## Features
- **Sticky navigation header** with the FoodSpot logo and links to each page section
- **Hero banner** with a call-to-action button leading to the menu
- **About Us section** in a two-column layout pairing text with imagery
- **Our Menu section** featuring six menu cards (burgers, pizzas, fries, sandwiches, tacos, and cold drinks) with hover effects
- **Kitchen Experts section** highlighting the restaurant's five featured chefs
- **Testimonials section** with customer photos and reviews
- **Contact form** with name, email, and message fields
- **Footer** with social media icons (Facebook, LinkedIn, Instagram) via Font Awesome
- **Responsive design** using Flexbox with a mobile breakpoint at 768px
- **Interactive touches** including card hover scaling and smooth CSS transitions

## Built With
- **HTML5** — semantic structure with clearly organized sections
- **CSS3** — Flexbox layouts, transitions, hover effects, and media queries
- **[Google Fonts](https://fonts.google.com/)** — [Inter](https://fonts.google.com/specimen/Inter) for body text and [Playwrite ID](https://fonts.google.com/specimen/Playwrite+ID) for decorative headings and the logo
- **[Font Awesome](https://fontawesome.com/)** — brand icons in the footer
- **GitHub Pages** — hosting and deployment

## Project Structure
```
ct-foodspot-website/
├── index.html      # Main page markup
├── styles.css      # All site styling
└── assets/         # Images for menu items, experts, and testimonials
```

## Deployment
The site is deployed with **GitHub Pages** directly from the `main` branch. Any changes pushed to `main` are automatically reflected on the live site.

## What I Learned
- Structuring a multi-section landing page with semantic HTML
- Building flexible, wrapping card layouts with Flexbox
- Creating a sticky header that stays visible while scrolling
- Making a layout responsive with media queries
- Integrating third-party resources (Google Fonts and Font Awesome) via CDN
